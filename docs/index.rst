***************************************
Welcome to PHASEfilter's documentation!
***************************************

**PHASEfilter is a software package that is possible to filter variants, SNPs and INDELs, that are present in heterozygous form, in phased genomes.**


Highlights
++++++++++

-  **filter SNPs and INDELs**;
-  **identify Loss of Heterozygous (LOH)**;
-  **create an alignment of two phased reference genomes**;
-  **possible to redefine the Allele Frequency threshold to redefine heterozygous form**;
